# Melon

> A plugin for the modern Minecraft server.

## Features

> More features are constantly being added to Melon!  
> Feel free to make a [feature request issue](https://github.com/dylanopen/Melon/issues)!

- MOTD manager - manage your server's description in the server list
  (fully customised with [MiniMessage](https://docs.papermc.io/adventure/minimessage/))
